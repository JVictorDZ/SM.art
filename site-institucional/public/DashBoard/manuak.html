Sensor Arduino:

  1. Desembale o kit e verifique se o conjunto do sensor se encontra em bom estado.
  2. Ligue o sensor e o aloque no mesmo ambiente em que se encontra o quadro ou obra.
  3. Após ligar o sensor, conecte-se a ele utilizando Bluetooth de um computador.

Servidor para Captura de Dados:

  1. Após conectar os sensores aos computadores, instale o NPM e NodeJS nas máquinas que serão utilizadas: 
    sudo apt-get update
sudo apt-get install npm
sudo apt-get install nodejs
  2. Com a instalação já feita, execute o comando NPM START através do Shell do sistema: 
    npm start
  3. Para averiguar se o servidor foi iniciado com sucesso, verifique o terminal onde o NPM foi executado e veja se há erros.

Configuração do Site:

  1. Acesse nosso site www.SMArt.com e você será direcionado para a tela inicial.
  2. Através do menu superior da página, acesse a tela de cadastro.
  3. Preencha os campos corretamente conforme o indicado.

Como Utilizar o Site:

  1. Após realizar o cadastro, você será direcionado para a tela de login, na qual deverá inserir as mesmas informações que cadastrou.
  2. Com o login feito, você será encaminhado para nossa tela de usuário, onde terá acesso a gráficos dos sensores, alertas sobre alterações de temperatura/umidade e também um sistema integrado com ChatBoot.
